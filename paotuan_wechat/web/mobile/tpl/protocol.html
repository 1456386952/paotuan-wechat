跑团助手平台活动发布协议

  1. 功能使用范围说明:
    2. 活动模块使用对象为跑团以及其他社会团体用于活动发布,报名,管理使用.
  2. 活动发布免责声明:
    3. 跑团助手为跑团运营与管理平台,在跑团活动发布后,活动发起人需要对该活动负责.使用本服务时,所发布的内容应符合网络道德，遵守中华人民共和国的相关法律法规。如果活动内容中有违反国家规定,损害国家利益,危害民族团结,扰乱社会秩序, 散布淫秽、色情、赌博、暴力、凶杀、恐怖或者教唆犯罪,侵害他人合法权益,以及其他行政法规命令禁止内容的,跑团助手有权对您发起的活动进行相应处理.
  3. 用户资料安全说明:
    4. 跑团助手为跑团运营与管理平台,不会将相关资料透露给第三方, 为保证交易安全，您应当妥善保管用户名和密码，因个人原因造成用户名和密码外泄、被盗等后果的，应自行承担由此所引发的一切损失。
  4. 提现说明:
    5. 如果您的活动存在收费情况,您收取的活动费用将会收取到跑团助手账号中,您可以在活动结束后通过发送邮件给hello@runningtogether.net方式申请提现,在完成审核后会进行相应的结算.
